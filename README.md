# دفتري — Daftari

**تطبيق ملاحظات عربي مصمم من الأساس لقارئ الشاشة NVDA** — بمفهوم Obsidian: ملاحظاتك ملفات Markdown عادية في مجلد تملكه أنت، مترابطة بروابط ويكي ووسوم وبحث شامل.

*An Arabic-first, NVDA-screen-reader-native note-taking app for Windows. Obsidian-style: your notes are plain Markdown files you own, connected with wiki links, tags, and full-text search. English UI available in Settings.*

## لماذا دفتري؟

معظم تطبيقات الملاحظات تُبنى للعين أولاً ثم "تُرقّع" للإتاحة. دفتري بُني بالعكس: كل ميزة صُممت لتعمل بالكامل من لوحة المفاتيح ويقرأها NVDA بوضوح، من التنقل بين الفقرات إلى تحرير الجداول.

## أبرز الميزات

- **ملفات Markdown عادية** في مجلد «القبو» — متوافقة مع Obsidian وأي محرر آخر، ولا احتكار لبياناتك.
- **روابط ويكي** `[[اسم الملاحظة]]`: اتباع بـ Ctrl+Enter، إدراج بالبحث Ctrl+K، وروابط واردة Ctrl+B.
- **محرر جداول شبكي** (Ctrl+Shift+G): أنشئ وحرر جداول Markdown في شبكة يقرؤها NVDA صفاً وعموداً كما في Excel — دون كتابة أي رمز.
- **معاينة HTML** (Ctrl+Shift+H): تفتح الملاحظة في المتصفح بعناوين وقوائم وجداول دلالية، وكل قسم قابل للطي والتوسيع.
- **تنقل صوتي ذكي**: بين الفقرات Ctrl+الأسهم (وصفوف الجداول تُقرأ خلايا نظيفة)، بين العناوين Alt+الأسهم، و«أين أنا؟» Ctrl+I يعلن موضعك كاملاً.
- **بحث شامل** Ctrl+Shift+F، **وسوم** Ctrl+T، **يوميات** Ctrl+D، **فتح سريع** Ctrl+P، **تنقل فوري بين ملاحظتين** Ctrl+Tab.
- **تراجع وإعادة متعددا المستويات**، حفظ تلقائي، والحذف إلى مجلد محذوفات قابل للاسترجاع.
- **نسخ احتياطي مضغوط** (Ctrl+Shift+B) إلى أي مجلد — ضعه داخل Google Drive أو OneDrive لمزامنة سحابية تلقائية.
- **واجهة ثنائية اللغة** (عربي/إنجليزي) وتنسيقات تاريخ متعددة منها أسماء الأشهر الجزائرية (جانفي، فيفري…).

اضغط **F1** داخل التطبيق لقائمة الاختصارات الكاملة.

## التحميل والتشغيل

لا يحتاج التطبيق تثبيتاً ولا أي متطلبات مسبقة:

1. حمّل أحدث إصدار من صفحة [Releases](../../releases).
2. فك الضغط وشغّل `Daftari.exe`.
3. عند أول تشغيل يُنشأ مجلد ملاحظاتك في `المستندات\دفتري`.

المتطلبات: ويندوز 10 أو 11 (64 بت).

## البناء من المصدر

```
dotnet build -c Release
```

يتطلب .NET 8 SDK. وللنشر كملف واحد مستقل:

```
dotnet publish -c Release -r win-x64 --self-contained true -p:PublishSingleFile=true
```

## التقنية

C#‎ / Windows Forms على .NET 8 — عناصر ويندوز الأصلية التي يتعامل معها NVDA بلا وسيط، مع إعلانات صوتية عبر UI Automation Notifications، وتحويل Markdown بمكتبة [Markdig](https://github.com/xoofx/markdig).
